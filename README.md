## Get started
```shell
git clone https://github.com/c-Bruno/I5-BE.git
cd I5-BE
```

### Install npm packages

Install the `npm` packages described in the `package.json` and verify that it works:
```shell
npm install
```

### Running the project
```shell
npm start
```

Shut it down manually with `Ctrl-C`.
